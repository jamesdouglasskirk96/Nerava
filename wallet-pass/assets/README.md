# Wallet Pass Assets

Place the following image files in this directory before building the pass:

## Required Files

### `logo.png`
- **Format**: PNG with transparent background
- **Usage**: Top-left logo displayed on the pass
- **Recommended size**: 160x50 pixels (or proportional)

### `icon.png`
- **Format**: PNG with transparent background
- **Size**: 29x29 pixels (exact size required)
- **Usage**: Small icon displayed on the pass

## Notes

- These files are not committed to the repository (binary assets)
- You must provide these files before running `../build.sh`
- The build script will validate that these files exist before proceeding
